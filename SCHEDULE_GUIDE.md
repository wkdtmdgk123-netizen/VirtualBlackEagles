# 비행 스케줄 관리 프로그램 사용 가이드

## 개요
Virtual Black Eagles 팀의 비행 일정을 관리하는 CLI 기반 프로그램입니다.

## 실행 방법

### 방법 1: Python으로 직접 실행
```bash
cd /Users/jangseungha/Documents/블랙이글홈페이지
python3 schedule.py
```

### 방법 2: 가상환경에서 실행
```bash
cd /Users/jangseungha/Documents/블랙이글홈페이지
.venv/bin/python schedule.py
```

## 주요 기능

### 1. 📋 일정 목록 조회
다양한 필터로 일정을 확인할 수 있습니다:
- **전체**: 모든 일정
- **다가오는 일정**: 오늘 이후 일정
- **지난 일정**: 과거 일정
- **오늘**: 오늘 진행되는 일정
- **이번 주**: 7일 이내 일정
- **이번 달**: 30일 이내 일정

각 일정에는 D-Day가 자동으로 표시됩니다.

### 2. ➕ 일정 추가
새로운 비행 일정을 등록합니다.

**필수 입력:**
- 제목
- 날짜 (YYYY-MM-DD 형식)

**선택 입력:**
- 장소
- 설명

**예시:**
```
제목: 연습 비행
장소: 공군 기지
날짜: 2025-12-15
설명: F-5 4기 편대비행 연습
```

### 3. 🔍 일정 검색
제목, 장소, 설명에서 키워드를 검색합니다.

**예시:**
```
검색어: 연습
검색어: 에어쇼
검색어: F-5
```

### 4. 📝 일정 상세 조회
ID를 입력하여 특정 일정의 모든 정보를 확인합니다.
- ID, 제목, 장소, 날짜
- 설명, 등록일, 수정일
- D-Day 및 상태 표시

### 5. ✏️ 일정 수정
기존 일정의 정보를 수정합니다.
- 수정하지 않을 항목은 Enter로 건너뜁니다.
- 날짜 형식: YYYY-MM-DD

### 6. 🗑️ 일정 삭제
일정을 영구적으로 삭제합니다.
- 삭제 전 확인 메시지 표시
- 되돌릴 수 없으므로 주의하세요.

### 7. 📊 통계
비행 스케줄 현황을 한눈에 확인합니다:
- 전체 일정 수
- 다가오는 일정
- 지난 일정
- 이번 주 일정
- 이번 달 일정

## 데이터베이스
- **위치**: `blackeagles.db`
- **테이블**: `schedules`
- 웹 관리자 페이지와 데이터 공유

## 주의사항

1. **날짜 형식**: 반드시 YYYY-MM-DD 형식으로 입력
2. **삭제**: 삭제한 일정은 복구 불가
3. **데이터베이스**: 웹사이트와 같은 DB 사용
4. **한글 지원**: 터미널이 UTF-8을 지원해야 함

## 단축키

- **Ctrl+C**: 프로그램 즉시 종료
- **Enter**: 다음으로 계속
- **0**: 정상 종료

## 트러블슈팅

### 문제: 한글이 깨져 보임
**해결**: 터미널 인코딩을 UTF-8로 설정
```bash
export LANG=ko_KR.UTF-8
```

### 문제: 데이터베이스 연결 실패
**해결**: blackeagles.db 파일이 있는지 확인
```bash
ls -la blackeagles.db
```

### 문제: 날짜 형식 오류
**해결**: YYYY-MM-DD 형식 사용
- ✅ 올바른 예: 2025-12-31
- ❌ 잘못된 예: 2025/12/31, 12-31-2025
