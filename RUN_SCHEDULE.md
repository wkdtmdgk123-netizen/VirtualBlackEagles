# 🚀 비행 스케줄 관리 프로그램 실행 가이드

## 📋 개요
이 프로그램은 **웹사이트와 완전히 독립적**으로 동작하는 CLI 기반 스케줄 관리 도구입니다.
- 독립 데이터베이스: `flight_schedules.db` (웹사이트의 `blackeagles.db`와 별도)
- 터미널에서 직접 실행
- D-Day 계산, 필터링, 검색, 통계 기능 제공

---

## 🔧 설치 및 실행 방법

### 방법 1: 가상환경 사용 (권장)

```bash
# 1. 프로젝트 폴더로 이동
cd /Users/jangseungha/Documents/블랙이글홈페이지

# 2. 가상환경 활성화
source .venv/bin/activate

# 3. 프로그램 실행
python schedule.py
```

### 방법 2: 전역 Python 사용

```bash
# 1. tabulate 설치 (최초 1회만)
pip3 install tabulate

# 2. 프로그램 실행
python3 schedule.py
```

### 방법 3: 실행 권한 부여 후 직접 실행

```bash
# 1. 실행 권한 부여 (최초 1회만)
chmod +x schedule.py

# 2. 직접 실행
./schedule.py
```

## 📚 주요 기능

### 1️⃣ 일정 목록 보기
```
일정 필터 옵션:
- 전체 일정
- 다가오는 일정 (D-Day 표시)
- 지난 일정
- 이번 주 일정
- 이번 달 일정
```

### 2️⃣ 새 일정 추가
```
입력 정보:
- 제목 (필수)
- 장소 (선택)
- 날짜 (YYYY-MM-DD 형식)
- 상세 설명 (선택)
```

### 3️⃣ 일정 검색
```
검색 가능 필드:
- 제목
- 장소
- 설명
```

### 4️⃣ 일정 수정
```
ID로 일정을 찾아 모든 정보 수정 가능
```

### 5️⃣ 일정 삭제
```
ID로 일정 삭제 (확인 메시지 표시)
```

### 6️⃣ 통계 보기
```
제공 정보:
- 전체 일정 수
- 다가오는 일정 수
- 지난 일정 수
- 가장 가까운 일정 (D-Day 포함)
```

## 💾 데이터베이스 위치

프로그램 실행 시 `flight_schedules.db` 파일이 자동으로 생성됩니다.

```
블랙이글홈페이지/
├── blackeagles.db          ← 웹사이트 DB (관리자 페이지에서 수정)
├── flight_schedules.db     ← CLI 프로그램 DB (이 프로그램에서 수정)
└── schedule.py
```

**⚠️ 주의:** 두 데이터베이스는 완전히 독립적이므로, 웹사이트 일정과 CLI 프로그램 일정은 서로 동기화되지 않습니다.

## 🎯 사용 시나리오

### 시나리오 1: 빠른 일정 확인
```bash
# 프로그램 실행
python schedule.py

# 1번 선택: 일정 목록 보기
# 2번 선택: 다가오는 일정 (D-Day 자동 계산)
# 0번 선택: 종료
```

### 시나리오 2: 새 비행 훈련 추가
```bash
# 프로그램 실행 → 2번 선택
제목: 편대 비행 훈련
장소: 서산 비행장
날짜: 2025-12-15
설명: 6기 편대 비행 및 기동 훈련
```

### 시나리오 3: 에어쇼 일정 검색
```bash
# 프로그램 실행 → 3번 선택
검색어: 에어쇼
# "에어쇼"가 포함된 모든 일정 표시
```

## 🔄 웹사이트와의 관계

| 구분 | 웹사이트 | CLI 프로그램 |
|------|----------|--------------|
| **데이터베이스** | blackeagles.db | flight_schedules.db |
| **관리 방법** | 관리자 페이지 (브라우저) | 터미널 명령어 |
| **접근 권한** | 로그인 필요 | 파일 접근 권한만 필요 |
| **용도** | 공식 홈페이지 일정 | 개인/팀 내부 일정 관리 |
| **D-Day 계산** | ❌ (기본 표시만) | ✅ (실시간 계산) |
| **필터링** | ❌ | ✅ (5가지 필터) |
| **검색** | ❌ | ✅ (키워드 검색) |
| **통계** | ❌ | ✅ (자동 집계) |

## 🛠 문제 해결

### ModuleNotFoundError: No module named 'tabulate'
```bash
# 가상환경에서 실행하거나
source .venv/bin/activate

# 또는 전역에 설치
pip3 install tabulate
```

### permission denied: ./schedule.py
```bash
# 실행 권한 부여
chmod +x schedule.py
```

### 데이터베이스 파일을 찾을 수 없습니다
```bash
# 프로그램이 자동으로 생성하므로 걱정 없음
# 처음 실행 시 flight_schedules.db가 자동 생성됩니다
```
